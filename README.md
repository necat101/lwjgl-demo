# LWJGL 3 Demo - Hello World

A basic LWJGL (Lightweight Java Game Library) 3 demonstration application.

**Based on:**
- HN Discussion: https://news.ycombinator.com/item?id=14797887
- LWJGL Homepage: https://www.lwjgl.org/

## Quick Start

```bash
# Clone the repository
git clone https://github.com/necat101/lwjgl-demo.git
cd lwjgl-demo

# Build with Maven
mvn clean package

# Run the demo
java -cp target/lwjgl-demo-1.0-SNAPSHOT.jar:target/lib/* com.necat.lwjgl.HelloLWJGL
```

## Overview

This project demonstrates LWJGL 3 basics with GLFW windowing and OpenGL context setup.

See README.md for full documentation.
